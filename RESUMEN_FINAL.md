## 🎓 DECISIONES ARQUITECTÓNICAS

### ✅ Constantes Centralizadas

```
PRO: Single source of truth, fácil mantenimiento, type-safe
CONTRA: Archivo podría crecer mucho (pero organizado)
SOLUCIÓN: Separar por archivo lógico (menu.ts, events.ts, etc.)
```

### ✅ Design Tokens + CSS Variables

```
PRO: Consistencia visual, cambios globales fáciles, accesibilidad
CONTRA: Requiere conocimiento de sistema de tokens
SOLUCIÓN: Documentación clara + ejemplos en componentes
```

### ✅ Schema.org JSON-LD

```
PRO: SEO mejorado, rich results en Google, estructura clara
CONTRA: Requiere validación
SOLUCIÓN: Usar herramientas de validación antes de deploy
```

### ✅ Helper Functions

```
PRO: DRY, reutilizable, fácil de testear
CONTRA: Más código inicial
SOLUCIÓN: Bien documentadas, con tipos TypeScript
```
